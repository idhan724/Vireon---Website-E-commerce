# Vireon - Website E-commerce

Vireon merupakan studi kasus implementasi arsitektur front-end modern menggunakan state management berbasis Zustand dan modular UI system dengan shadcn/ui. Website ini dibangun menggunakan **React**, **TypeScript**, dan **Vite**.  
Project ini menampilkan fitur daftar produk, filter kategori, pencarian produk, manajemen keranjang belanja, serta halaman checkout dengan struktur komponen yang rapi dan reusable.

## Live Demo

## 🚀 Fitur Utama

- ✅ Menampilkan daftar produk berdasarkan kategori (Tech, Fashion, Home & Living, dll)
- 🔍 Pencarian produk secara real-time
- 🛒 Fitur tambah ke keranjang
- ➕➖ Penambahan dan pengurangan jumlah produk
- ☑ Pilih item satuan atau pilih semua di keranjang
- 💰 Perhitungan total harga otomatis
- 🌙 Mode Dark / Light
- 📦 Halaman checkout
- 💵 Format harga dalam Rupiah
- ⚡ State management menggunakan Zustand
